O try-catch é uma estrutura utilizada em C# para tratamento de exceções.
O bloco try executa um código que pode gerar erros, enquanto o bloco catch captura essas exceções, evitando que o programa seja encerrado inesperadamente.

No projeto, o try-catch foi utilizado como estratégia de prevenção de erros, permitindo que entradas inválidas ou falhas sejam tratadas de forma controlada, exibindo mensagens ao usuário e mantendo o sistema funcionando corretamente. E foi o que aconteceu no exemplo do segundo print, ao invés do programa crashar, ele assimilou o erro, enviou um feedback para o usuário, lhe informando do erro e a maneira apropriada de executar a tarefa.
